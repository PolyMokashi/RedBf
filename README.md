## Project Setup
### 1. Clone the repository:
```bash
git clone https://github.com/PolyMokashi/RedBf.git
cd RedBf
```

### 2. Install dependencies:
```bash
npm install
```

### 3. Run the development server:
```bash
npm run serve
```

### 4. Build for production:
```bash
npm run build
```

### 5. Lint and fix files:
```bash
npm run lint
```
